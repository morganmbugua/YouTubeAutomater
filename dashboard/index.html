🚀 First-Time Setup Steps

  1. Create a public GitHub repository and upload all files from the zip
  2. Enable GitHub Pages: Settings → Pages → Deploy from branch → main → /dashboard
  3. Get all API keys listed above (at minimum: Anthropic + YouTube OAuth)
  4. Run python scripts/auth_youtube.py on your computer to get YouTube tokens
  5. Add all secrets: Settings → Secrets and variables → Actions
  6. Test: Actions → YT Autopilot — Generate & Upload Videos → Run workflow → num_videos=1
  7. Watch the Actions log — if it succeeds, the cron schedule takes over automatically
  8. Your dashboard: https://YOUR_USERNAME.github.io/REPO_NAME
