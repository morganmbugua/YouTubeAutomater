🚀 Setup Steps

  1. Create a public GitHub repo and upload all files
  2. Rename dashboard/ folder to docs/
  3. Enable GitHub Pages: Settings → Pages → main branch → /docs
  4. Get free Groq key at console.groq.com
  5. Set up Google Cloud: enable YouTube Data API v3 + Analytics API
  6. Get free Pexels key at pexels.com/api
  7. Run python scripts/auth_youtube.py for OAuth tokens
  8. Add all secrets: repo Settings → Secrets → Actions
  9. Test: Actions → Generate Videos → Run workflow → num_videos=1
  10. Verify YouTube channel at youtube.com/verify for thumbnails
