🚀 Setup Steps

  1. Create a public GitHub repo and upload all files
  2. Enable GitHub Pages: Settings → Pages → main → /docs
  3. Get free Gemini key at aistudio.google.com
  4. Set up Google Cloud project, enable YouTube Data API v3
  5. Get free Pexels key at pexels.com/api
  6. Run python scripts/auth_youtube.py for OAuth tokens
  7. Add all secrets: repo Settings → Secrets → Actions
  8. Test: Actions → Generate Videos → Run workflow → num_videos=1
